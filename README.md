# Paris Velib Map
[ReactJS] Training with Velib API in Paris area

# Features
- ReactJS
- Bulma CSS
- Webpack
- React Google Maps (with Clusters)
- fetch with Promise

/!\ This repo does **not** use Redux for lifecycle state management.
